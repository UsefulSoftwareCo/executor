## Trigger Types And Event Ingestion

Definition-level trigger kinds are `event`, `schedule`, `poll`, and `manual` in `/home/dana/projects/augment/agents/executor/apps/open-agents/lib/automation/types.ts`. `event` triggers match by optional `source` and required `type`; `schedule` and `poll` are converted to events by the scheduler; `manual` matches only test/manual event pathways.

It also checks `CRON_SECRET`, calls `ensureSourcePollingWorkflow()`, and returns a start result in `/home/dana/projects/augment/agents/executor/apps/open-agents/app/api/source-polling/route.ts`.
